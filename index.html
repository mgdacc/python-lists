¿Qué es una Lista?

Una colección ordenada y modificable de elementos. Permite duplicados.

2 / 16

Crear una Lista

Se define con corchetes `[]`, separando los elementos con comas.

# Lista de números
numeros = [1, 2, 3, 4, 5]

# Lista de strings
frutas = ["manzana", "banana", "cereza"]
3 / 16

Acceder a Elementos (Índice)

Se accede a un elemento por su posición, comenzando desde 0.

frutas = ["manzana", "banana", "cereza"]
print(frutas[0]) # 'manzana'
print(frutas[1]) # 'banana'
4 / 16

Índice Negativo

Permite acceder a los elementos desde el final de la lista.

frutas = ["manzana", "banana", "cereza"]
print(frutas[-1]) # 'cereza'
print(frutas[-2]) # 'banana'
5 / 16

Slicing (Rebanado)

Permite obtener una sub-lista especificando un rango de índices.

numeros = [0, 1, 2, 3, 4, 5]
print(numeros[2:5]) # [2, 3, 4]
print(numeros[:3])  # [0, 1, 2]
6 / 16

Modificar un Elemento

Se puede cambiar el valor de un elemento usando su índice.

frutas = ["manzana", "banana", "cereza"]
frutas[1] = "mango"
print(frutas) # ['manzana', 'mango', 'cereza']

Método: .append()

Añade un elemento al final de la lista.

frutas = ["manzana", "banana"]
frutas.append("naranja")
print(frutas) # ['manzana', 'banana', 'naranja']
8 / 16

Método: .insert()

Añade un elemento en una posición específica.

frutas = ["manzana", "cereza"]
frutas.insert(1, "banana")
print(frutas) # ['manzana', 'banana', 'cereza']
9 / 16

Método: .pop()

Elimina y devuelve el último elemento (o el de un índice específico).

frutas = ["manzana", "banana", "cereza"]
fruta_eliminada = frutas.pop()
print(fruta_eliminada) # 'cereza'
print(frutas) # ['manzana', 'banana']
10 / 16

Método: .remove()

Elimina la primera aparición de un valor específico.

frutas = ["manzana", "banana", "manzana"]
frutas.remove("manzana")
print(frutas) # ['banana', 'manzana']
11 / 16

Método: .sort()

Ordena la lista de forma ascendente (alfabética o numérica).

numeros = [4, 1, 8, 2]
numeros.sort()
print(numeros) # [1, 2, 4, 8]
12 / 16

Método: .reverse()

Invierte el orden de los elementos de la lista.

frutas = ["manzana", "banana", "cereza"]
frutas.reverse()
print(frutas) # ['cereza', 'banana', 'manzana']
13 / 16

Función: len()

Devuelve el número de elementos que contiene la lista.

frutas = ["manzana", "banana", "cereza"]
print(len(frutas)) # 3
14 / 16

Operador: in

Verifica si un elemento existe dentro de la lista.

frutas = ["manzana", "banana", "cereza"]
if "banana" in frutas:
    print("Sí, tenemos bananas.")
15 / 16

Recorrer una Lista (Loop)

Se usa un ciclo `for` para ejecutar una acción en cada elemento.

for fruta in ["manzana", "banana", "cereza"]:
    print(f"Me gusta comer {fruta}")
